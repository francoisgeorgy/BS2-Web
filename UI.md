# User Interface

The UI will try to emphasize the signal flow (sound path). It is not important to reproduce the layout of the hardware BS2. 

## Main parts

- Oscillators
    - Osc. 1
    - Osc. 2
    - Sub osc.
- Low Frequency Oscillators (LFO)
- Mixer
- Filter
- Envelopes
    - Amp envelope
    - Mod envelope
- Effects
- Portamento
- Arpeggiator
